Odaberite folder gdje cete smjestiti projekat

u tom folderu na prazno mjesto desni klik kliknite git bash
ako ne vidite tu opciju onda preko start menu otvorite git bash

kucajte
git clone https://github.com/Nokee/SmartHouseOOAD.git

to bi trebalo skinuti projekat u vas folder


ukoliko ste kreirali file ili izmenili neki file potrebno j e u git bash-u ukucati

git add ime_file
git commit -m "Text kojima cete reci sta ste uradili u tom filu"

da bi ste uplodovali na github to potrebna je ukucati

git push

bit ce vam zatrazen username i password ukucajte

da bi ste skinuli promjene sa github ukucajte

git pull

jos jedna korisna naredba
je 

git status

koja ce vam reci stanje vaseg foldera
crveni fileovi nisu dodani treba git add ko gore
zeleni treba git commit -m "obavezno tekst "
Obavezno u git commit dodavat ovo -m zasto? zato
kada otvorite git bash sljedeci but obavezni git pull prvo mislim bilo bi pozeljno
